# 🛒 Shopping Website

A modern and interactive shopping website built with **HTML, CSS, and Vanilla JavaScript**. Users can browse products, add them to the cart, adjust quantities, remove items, and see the total price in real-time.

## ✨ Features

- Beautiful product cards with hover effects
- Add items to cart with one click
- Fully functional shopping cart:
  - Increase / Decrease item quantity
  - Remove items completely
  - Real-time total price calculation
- Responsive and clean UI design
- Modular JavaScript architecture (ES6 Modules)
- Data loaded from JSON file

## 🛠️ Technologies

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Box Shadow)
- **Vanilla JavaScript** (ES6+)
- No external frameworks or libraries
